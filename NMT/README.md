# 인공신경망 기반 기계번역 작업 폴더
## 설명
서식에서 추출한 텍스트 데이터를 변환해주는 NMT 관련 작업 폴더

## 소스설명
### bert-nmt
bert nmt 모델(local에서 진행하다 보니 gpu가 없어서 모델은 나왔으나 활용하지 못함)

### ko_en_neural_machine_translation
원 기능은 한국어-영문 기계번역으로 mxnet gluon 활용한 번역으로 태국어로 변경해서 사용
(실질적인 모델 활용)

### simple-nmt
간단한 nmt 모델로 활용

### tltk_pythainlp.ipynb
태국어 전처리 패키지
